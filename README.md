# Dobby — Polymarket Advisor Bot

A Telegram bot that pulls Polymarket crypto markets and provides short, witty LLM insights.

### Setup
1. Copy `.env.example` → `.env`
2. Add your tokens:
```
TG_TOKEN=your_telegram_token
LLM_API_KEY=your_llm_api_key
```
3. Install deps:
```
npm install
```
4. Run locally:
```
npm start
```
